1. Что такое регулярные выражения и где они эффективны?
Регулярные выражения (RegEx) — это шаблоны для поиска, извлечения и замены текста по заданным правилам. Эффективны для: валидации данных (email, телефон), парсинга логов, очистки текста, массового поиска/замены, извлечения структурированных данных из неструктурированного текста.
2. Какой модуль импортировать для работы с RegEx в Python?
Модуль re: import re
3. Разница между re.search() и re.findall()?
re.search() находит первое совпадение и возвращает объект Match (или None). re.findall() находит все непересекающиеся совпадения и возвращает их списком строк. Для получения всех совпадений используется findall().
4. Как проверить, что строка начинается с паттерна?
Использовать re.match(pattern, string) — ищет совпадение только в начале строки. Альтернатива: добавить ^ в начало паттерна и использовать re.search().
5. Что означает метасимвол . (точка)?
Точка . соответствует любому одному символу, кроме новой строки (\n). Например, паттерн a.c совпадёт с "abc", "a c", "a@c", но не с "ab\nc".
6. Что такое группировка и какой у неё синтаксис?
Группировка позволяет выделять части совпадения для извлечения или обратных ссылок. Синтаксис: круглые скобки (...). Пример: r'(\d{2})\.(\d{2})' создаёт две группы — день и месяц. Доступ к группам: через match.group(1), match.group(2) или в re.sub() через \1, \2.
